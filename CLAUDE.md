# CLAUDE.md — 子平命理分析系统

## 项目概述

纯 Python 实现的子平命理（八字）分析引擎，100% 离线运行，无任何外部依赖。

## 核心文件

| 文件 | 职责 |
|------|------|
| azi_engine.py | 规则引擎核心——八字排盘、十神推算、五行生克、格局判定、调候用神等 |
| azi_analyzer_offline.py | CLI 交互界面，调用 bazi_engine 输出完整命局报告 |

## 关键概念（供 AI 理解）

- **四柱**：年柱、月柱、日柱、时柱，每柱一天干一地支，共八个字
- **十神**：比肩、劫财、食神、伤官、偏财、正财、七杀、正官、偏印、正印——由日干与其他七字生克关系推导
- **五行**：木、火、土、金、水；天干地支各有所属
- **地支藏干**：每个地支暗藏 1-3 个天干（如子藏癸、丑藏己癸辛）
- **月令**：月支本气，决定命局基调，是取用神的核心依据
- **调候用神**：寒湿喜暖燥、热燥喜寒湿——源于《穷通宝鉴》
- **格局**：
  - 普通命局：月令为用，配以月干十神成格
  - 从旺格：日干极弱，全局生助日干
  - 从势格：日干无根，全局克泄耗日干
- **大运/流年**：大运十年一步，流年逐年推进，分析冲合会刑穿

## 技术规范

- Python 3 标准库，无第三方依赖
- azi_engine.py 为纯计算模块，可被其他程序 import：
  `python
  from bazi_engine import BaziCalculator, BaziAnalyzer, get_shishen
  `
- 涉及命理古籍经典逻辑时，引用依据：
  - 《滴天髓》——五行旺衰判断
  - 《渊海子平》——十神体系、格局理论
  - 《三命通会》——神煞、大运流年
  - 《穷通宝鉴》——调候用神、月令喜忌

## 代码风格

- 变量命名使用中文拼音或英文，避免混淆（如 
ian_zhu / day_gan）
- 核心规则模块（bazi_engine.py）不包含任何 UI 或 I/O 逻辑
- CLI 界面（bazi_analyzer_offline.py）仅做输入解析与格式化输出

## 修改规则

- **禁止随意改动排盘逻辑**：八字排盘基于万年历节气推算，修改需参考《子平命理》排月柱、日柱规则
- **十神推算须严格遵循五行生克关系**：日干为基准，阳克阳/阴克阴为七杀，阳克阴/阴克阳为正官，以此类推
- **新增功能需标注依据**：引用哪本经典、何种说法
