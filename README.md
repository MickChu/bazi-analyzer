# 子平命理实战派宗师

八字命理分析系统，基于《滴天髓》《渊海子平》《三命通会》《穷通宝鉴》四大经典，纯离线运行，无需任何外部 API。

## 项目结构

`
bazi_analyzer_offline.py  # CLI 交互界面（直接运行）
bazi_engine.py            # 核心规则引擎（可被其他程序 import）
`

## 安装依赖

项目无外部依赖，纯 Python 3 标准库实现。

`ash
# 任意 Python 3 环境即可运行
python bazi_analyzer_offline.py
`

## 功能特性

- **八字排盘**：输入出生时间，自动排出年柱、月柱、日柱、时柱
- **十神系统**：自动推算各柱十神（比肩、劫财、食神、伤官、正财、偏财、正官、七杀、正印、偏印）
- **五行分析**：天干地支五行属性、地支藏干、月令深浅
- **格局判定**：普通命局、从旺命局、从势命局等
- **大运推演**：十年一步大运，自动排布
- **流年分析**：逐年纪柱、五行、冲合会刑穿
- **用神选取**：结合《穷通宝鉴》调候用神与扶抑用神

## 使用方法

`ash
python bazi_analyzer_offline.py
`

按提示输入性别、出生日期（公历/农历）、出生时间，即可获得完整命局分析报告。

## 设计理念

- **直断为本**：重结构、重五行流通，轻神煞
- **完全离线**：零依赖、零网络请求
- **经典为据**：所有规则来源于子平古籍经典
